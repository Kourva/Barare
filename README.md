

# Barare
#### Kivy android app with login page | You can use it and add your functions !

+ 6 screens
+ Kmap solver (for test)
+ Login page
+ Backgroud image

# Install
+ clone
```bash
git clone https://github.com/Kourva/Barare
```
+ requiremets
```bash
pip install kivy
```
+ run
```bash
python3 main.py
```
